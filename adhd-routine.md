# SOS's ADHD-Friendly Daily Routine

> Rafa says: Keep it simple. Keep it consistent. No fluff.

---

## 🌅 MORNING BLOCK (6:30 - 9:00)

### 6:30 AM - Wake Up Protocol
- [ ] ☀️ Wake up (same time daily, even weekends)
- [ ] 💧 Drink 500ml water immediately (hydrate brain)
- [ ] 💊 **Morning meds & supplements** (see tracker below)
- [ ] 🚶‍♂️ 5-min movement/stretch (jumpstart dopamine)

### 7:00 AM - Breakfast + Setup
- [ ] 🍳 High-protein breakfast (30g+ protein stabilizes meds)
- [ ] 📱 **Phone check LIMIT**: 10 min max (scroll trap risk)
- [ ] 📝 Review today's top 3 priorities (write them down)

### 7:30 AM - First Focus Block (Deep Work)
- [ ] **Target**: PhD Chapter work OR PHPS40600 study
- [ ] Duration: 90 min (Pomodoro: 25+5, 25+5, 20)
- [ ] No phone. No email. Just focus.

### 9:00 AM - Movement Break
- [ ] 🏃‍♂️ 10-15 min walk or light exercise
- [ ] 💧 Hydrate
- [ ] 🧠 Brain reset before next block

---

## 🧠 MID-DAY BLOCK (9:00 - 13:00)

### 9:30 AM - Second Focus Block
- [ ] **Target**: Opposite of morning (if PhD→study, if study→PhD)
- [ ] Duration: 75 min

### 10:45 AM - Snack + Supplement
- [ ] 🥗 Light snack (fruit + nuts)
- [ ] 💊 Mid-morning supplement (if applicable)
- [ ] ☀️ 5 min outside/natural light (circadian rhythm)

### 11:00 AM - Admin/Email Block (30 min)
- [ ] 📧 Process email inbox (quick: reply/delegate/archive)
- [ ] 📋 Check calendar for today + tomorrow
- [ ] **Then close email** — don't live in inbox

### 11:30 AM - Third Focus Block
- [ ] **Target**: Deep work or research
- [ ] Duration: 60 min

### 12:30 PM - Lunch + Reset
- [ ] 🍲 Proper lunch (carbs + protein + veggies)
- [ ] 🚶‍♂️ 15-min walk post-lunch (blood sugar stability)
- [ ] **NO PHONE during lunch** (mindful eating)

---

## ☀️ AFTERNOON BLOCK (13:00 - 17:30)

### 13:30 PM - Fourth Focus Block
- [ ] **Target**: Writing, analysis, or creative work
- [ ] Duration: 60 min
- [ ] If meds wearing off → switch to lighter tasks

### 14:30 PM - Movement Break
- [ ] 🏋️‍♂️ Exercise block (20-30 min)
- [ ] Options: gym, home workout, brisk walk
- [ ] Dopamine boost → better evening focus

### 15:00 PM - Flex Block
- [ ] **Target**: Meetings, calls, collaborative work
- [ ] Or: Administrative tasks, planning
- [ ] Duration: 60-90 min

### 16:30 PM - Late Afternoon Check
- [ ] ✅ Review today's progress
- [ ] 📝 Identify what's left → prioritize or defer
- [ ] 💊 Afternoon supplements (if applicable)

### 17:00 PM - Wind-Down Starts
- [ ] No new deep work after 5 PM
- [ ] Light tasks only: reading, organizing, emails
- [ ] Prepare brain for evening

---

## 🌙 EVENING BLOCK (17:30 - 22:00)

### 18:00 PM - Dinner + Supplements
- [ ] 🍽️ Nutrient-dense dinner
- [ ] 💊 Evening supplements (see tracker)
- [ ] 🚶‍♂️ 10-min post-dinner walk (digestion + mental reset)

### 19:00 PM - Prep for Tomorrow (CRITICAL)
- [ ] 📝 Write tomorrow's top 3 priorities (TONIGHT)
- [ ] 👕 Lay out clothes/gym gear
- [ ] 🎒 Pack bag for tomorrow
- [ ] 💊 Prep morning supplements (pill sorter)
- [ ] **Why:** Reduces decision fatigue tomorrow morning

### 20:00 PM - Low-Dopamine Block
- [ ] 📖 Reading (non-digital preferred)
- [ ] No social media (scroll trap)
- [ ] Relaxing hobby or conversation

### 21:00 PM - Digital Sunset
- [ ] 📱 Phone on charger (outside bedroom if possible)
- [ ] 💻 Laptop closed
- [] Dim house lights (melatonin production)

### 21:30 PM - Sleep Protocol
- [ ] 🛌 In bed by 21:30
- [ ] 📖 Light reading only (no screens)
- [ ] 🧘‍♂️ 5-min breathing or meditation
- [ ] Lights out by 22:00

## ⚡ ADHD-SPECIFIC RULES

### 1. TIME BLOCKS ARE NON-NEGOTIABLE
- Focus blocks = deep work only
- Breaks = actual breaks (no phone workarounds)
- Same times every day = brain learns rhythm

### 2. TASK MANAGEMENT
- Never more than 3 priorities per day
- Write them down (brain dump → trust paper)
- If stuck → 2-min rule: just start for 2 minutes

### 3. PHONE HYGIENE
- No phone in bedroom (buy alarm clock if needed)
- Set time limits on addictive apps
- Check emails only during designated blocks

### 4. MOVEMENT IS MEDICINE
- Exercise → dopamine → better focus
- Walks = free mental health tool
- Post-meal walks = blood sugar stability

### 5. SLEEP IS FOUNDATION
- 7.5-8 hours non-negotiable
- Consistent wake time > consistent bedtime
- Poor sleep = ADHD symptoms × 2
